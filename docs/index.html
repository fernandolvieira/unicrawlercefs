Calendário Instituições Por estado Como isto funciona

O Unicrawler é a ferramenta do Colégio Frei Seráfico (CEFS) para acompanhar, em um só lugar, o calendário, os links oficiais e os documentos exigidos dos principais processos seletivos — seriados ou convencionais — de 20 instituições públicas e particulares de MG, RJ e SP, incluindo o Afya São João del-Rei (ex-UNIPTAN).

04 — Metodologia

Como este painel é mantido

Crawler automático

Um robô em Python roda periodicamente via GitHub Actions, visita o site oficial de cada processo seletivo e procura links para PDFs de editais e trechos de texto que combinem datas com palavras como "inscrições", "prova" ou "resultado".

Curadoria manual

Datas marcadas como confiança alta foram confirmadas manualmente em fontes oficiais. O que vem só do crawler aparece marcado como extração automática — útil como pista, não como certeza.

Como contribuir

O projeto é público no GitHub. Para corrigir um link, atualizar uma data ou adicionar uma instituição, edite os arquivos em data/ e abra um Pull Request.

Unicrawler — ferramenta de apoio do Colégio Frei Seráfico (CEFS). Painel comunitário e não-oficial: as informações finais sempre prevalecem nos editais publicados pelas próprias instituições. Verifique sempre a fonte oficial antes de tomar decisões.
